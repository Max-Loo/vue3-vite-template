# vue3-vite-模板

## 开始
```bash
# 安装依赖
pnpm i

# 运行开发环境
pnpm run dev

# 预览打包
pnpm run preview

# 打包
pnpm run build

# 校验格式，git commit 的时候也会校验一次
pnpm run lint
```

## 技术栈
* [vue3](https://v3.vuejs.org/)
* [vite](https://cn.vitejs.dev)
* [typescript](https://www.typescriptlang.org/)
* [pnpm](https://www.pnpm.cn/)

## vscode 必要插件
* Vue Language Features (Volar)
